# 港台大不同之食物用語篇

### 煮食方法

| 香港用語 | 台灣用語 | 英文 |
|:---: | :---: | 
| 煎 | 煎 | Pan Fry |
| 炒 | 炒 | Stir-fry |
| 蒸 | 蒸 | Steam |
| 炸 | 炸 | Deep fry |
| 炆 | （粵菜手法） | Braise |
| 烚 | 水煮 | Boil |
| 汆水/飛水/焯 | 川燙 | Blanch/ parboil |
